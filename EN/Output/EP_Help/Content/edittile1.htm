Edit an Existing Tile

Tiles set for a generic time frame, such as Last Month or Last Year, update automatically. However, there may be occasion to add or remove parameters of a given tile which currently displays on a dashboard.

NOTE: When you edit a tile, you are actually cloning the selected tile into the work environment it was built in where you can make the desired revisions. The selected tile remains on the dashboard.

To view a dashboard, click the Dashboard icon () on the side navigation and then click the dashboard title on the Dashboards sidebar. Users having access to Tools may also view a dashboard from the Dashboard Manager by clicking the View icon ().
