Building a New Dashboard with Dashboard Manager

To view the Dashboard Manager, click the Tools icon () on the side navigation, and select Dashboard Manager from the Tools sidebar. The Dashboard Manager page displays a list of dashboards.

Click Create New Dashboard. The Create a New Dashboard dialog displays with several layout templates and a text field to type the name of the new dashboard.
