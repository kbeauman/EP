Uma Pontuação é um valor numérico que varia de zero a cem e que representa uma medida específica. O Portal Executivo: oferece os seguintes tipos de pontuações:

Resumo de múltiplos canais

Isto é duas ou mais pontuações apresentadas em um gráfico de rosca ou grade tipo planilha de cálculo. (missing or bad snippet)

Modo de exibição de gráfico de rosca

Conselho: Passe o ponteiro do mouse sobre os elementos do gráfico para exibir pontuações menores. As etiquetas podem ficar ocultas quando várias pontuações têm valores mais baixos. Elas podem ser exibidas posicionando-se o ponteiro do mouse sobre cada elemento do gráfico.

Essa visualização interativa mostra a pontuação de cada canal comum minigráfico e a pontuação combinada no centro. Clique em cada um dos seguintes elementos para exibir os dados mais detalhados:

  • Channel Score Label - omposto do canal, pontuação, e Contagem n, esse elemento exibe dados de Resumo, dosElementosComportamentos futuros que compõem a medida.
  • Minigráfico - Exibe um gráfico de linhas com pontos de dados da pontuação total durante o intervalo de datas selecionado, juntamente com dados de Resumo que incluem a Pontuação, a Ponderação,e a Contagem n.
    NOTA: Minigráficos com duração inferior a 60 dias são mostrados em incrementos diários. Para períodos com mais de 60 dias, os minigráficos são mostrados em incrementos semanais.
  • Pontuação combinada - Composto do nome, pontuação, e Contagem n, esse elemento apresenta as mesmas informações que o minigráfico..

Exibição de grade

Esta tabela interativa oferece os seguintes recursos:

  • Pontuação combinada - Clique no cabeçalho da tabela para exibir dados de Resumo que incluem a Pontuação, aPonderação, e a Contagem n.
  • Classificar por coluna - Clique no cabeçalho de qualquer coluna para classificar os dados com base nos valores selecionados da coluna, com o menor valor primeiro. Clique novamente para inverter a ordem.
  • Pontuação do canal - Clique em cada item de linha para exibir os dados de Resumo, dosElementos, e dos Comportamentos futuros que compõem a medida.

Os gráficos de pizza são baseados na Contagem n, ou seja, cada fatia está dimensionada para refletir a distribuição percentual das respostas que têm uma classificação específica, com a menor Contagem n no topo e percentagens crescentes progredindo no sentido horário. (missing or bad snippet) Cada seção (se houver espaço) é identificada com a resposta à pergunta, a pontuação e o percentual de respostas (Contagem n). Na imagem a seguir:

  • Resposta à pergunta = 5
  • Pontuação = 56
  • Percentual de respostas = 3%
  • Contagem n = 1

No caso de haver várias respostas com Contagem n baixa Contagem n, a etiqueta de cada uma delas só aparece quando o ponteiro do mouse é passado sobre o gráfico de pizza. Além disso, quando a resposta a uma pergunta estiver truncada, clique na etiqueta ou na seção de pizza para exibir toda a resposta na barra lateral.

Exibição de grade

As classificações são ordenadas segundo as opções de resposta para a pergunta selecionada. Por exemplo, uma pergunta com opções de resposta Sim, Não e Não tenho certeza mostra as classificações nessa mesma sequência. No entanto, é possível classificar por cabeçalho de coluna. Clique uma vez no cabeçalho da coluna desejada para classificar em ordem descendente (maior para menor) e duas vezes para classificar em ordem ascendente (menor para maior).

Cada modo de exibição É interativo para exibir um minigráfico e dados de Resumo para cada item clicado.

Resumo das pontuações

O bloco“Resumo das pontuações” exibe um valor numérico de cada medida para fornecer uma recapitulação rápida de uma pontuação específica ou de todas as pontuações de um intervalo de tempo definido. As pontuações são organizadas de acordo com as categorias dos Elements (também conhecidos comoFatores geradores de satisfação), SatisfaçãoeComportamentos futuros (como é mais provável que um visitante responda como resultado da experiência).

As pontuações são apresentadas como uma grade ou células individuais, com cada pontuação exibindo um minigráfico na barra lateral direita quando selecionada. As grades podem ser temporariamente classificadas por coluna em ordem ascendente ou descendente, bastando para isso clicar no cabeçalho da coluna. No entanto, a ordem de classificação padrão é reaplicada quando a sessão é encerrada.
