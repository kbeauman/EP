Os seguintes modos de exibição estão disponíveis...

Exibição de gráfico de pizza

Conselho: Passe o ponteiro do mouse sobre os elementos do gráfico para exibir pontuações menores. As etiquetas podem ficar ocultas quando várias pontuações têm valores mais baixos. Elas podem ser exibidas posicionando-se o ponteiro do mouse sobre cada elemento do gráfico.

Essa visualização interativa mostra a pontuação de cada canal com um minigráfico e a pontuação combinada no centro. Clique em cada um dos seguintes elementos para exibir os dados mais detalhados:

  • Etiqueta de pontuação do canal - Composto do canal, pontuação, e Contagem n, esse elemento exibe dados de Resumo, dos Elementos e dos Comportamentos futuros que compõem a medida.
  • Minigráfico - Exibe um gráfico de linhas com pontos de dados da pontuação total durante o intervalo de datas selecionado, juntamente com dados de Resumo que incluem a Pontuação, a Ponderação e a Contagem n.
    NOTA: Minigráficos com duração inferior a 60 dias são mostrados em incrementos diários. Para períodos com mais de 60 dias, os minigráficos são mostrados em incrementos semanais.
  • Pontuação combinada - Composto do nome, pontuação, e Contagem n, esse elemento apresenta as mesmas informações que o minigráfico.

Exibição de grade

Esta tabela interativa oferece os seguintes recursos:

  • Pontuação combinada - Clique no cabeçalho da tabela para exibir dados de Resumo que incluem a Pontuação, a Ponderação e a Contagem n.
  • Classificar por coluna - Clique no cabeçalho de qualquer coluna para classificar os dados com base nos valores selecionados da coluna, com o menor valor primeiro. Clique novamente para inverter a ordem.
  • Pontuação do canal - Clique em cada item de linha para exibir os dados de Resumo, dos Elementos e dos Comportamentos futuros que compõem a medida. A divisa dupla do Resumo exibe a página do Portal Analítico correspondente.

Comparação de respostas

As pontuações são compiladas com base em diversos critérios selecionadosnos menus das barras laterais Configurações e Dados. Por padrão, o painel de exibição mostra um bloco representando os dados do dia anterior para o primeiro item de cada menu. As pontuações são mostradas em um gráfico de pizza com as informações básicas no cabeçalho do bloco. Clique na divisa acima do bloco para mostrar ou ocultar dados adicionais. Selecione um intervalo de datas e os itens de menu desejados nas barras laterais Seleção de dados e Configurações.

Os seguintes modos de exibição estão disponíveis...
  • Gráfico de pizza - Os gráficos de pizza são baseados na Contagem n, ou seja, cada fatia está dimensionada para refletir a distribuição percentual das respostas que têm uma classificação específica, com a menor Contagem n no topo e percentagens crescentes progredindo no sentido horário. Cada seção (se houver espaço) é identificada com a resposta à pergunta, a pontuação e o percentual de respostas (Contagem n). Na imagem a seguir:
    • Resposta à pergunta = 5
    • Pontuação = 56
    • Percentual de respostas = 3%
    • Contagem n = 1

    No caso de haver várias respostas com Contagem n baixa, a etiqueta de cada uma delas só aparece quando o ponteiro do mouse é passado sobre o gráfico de pizza. Além disso, quando a resposta a uma pergunta estiver truncada, clique na etiqueta ou na seção de pizza para exibir toda a resposta na barra lateral.

  • Gráfico de barras verticais e gráfico de barras horizontais - Gráficos de barras são baseados na Contagem n, ou seja, a altura/comprimento de cada barra representa a porcentagem de respostas. As classificações são ordenadas segundo as opções de resposta para a pergunta selecionada. Por exemplo, uma pergunta com opções de resposta Sim, Não e Não tenho certeza mostra as classificações nessa mesma sequência. (missing or bad snippet)
  • Grid - As classificações são ordenadas segundo as opções de resposta para a pergunta selecionada. Por exemplo, uma pergunta com opções de resposta Sim, Não e Não tenho certeza mostra as classificações nessa mesma sequência. No entanto, é possível classificar por cabeçalho de coluna. Clique uma vez no cabeçalho da coluna desejada para classificar em ordem descendente (maior para menor) e duas vezes para classificar em ordem ascendente (menor para maior).

Cada modo de exibição é interativo para exibir um minigráfico e dados de Resumo para cada item clicado.

Resumo das pontuações

O bloco Resumo das pontuações exibe um valor numérico de cada medida para fornecer uma recapitulação rápida de uma pontuação específica ou de todas as pontuações de um intervalo de tempo definido. As pontuações são organizadas de acordo com as categorias dosElementos (também conhecidos como Fatores geradores de satisfação), Satisfaçãoe Comportamentos futuros (como é mais provável que um visitante responda como resultado da experiência).

As pontuações são apresentadas como uma grade () ou células individuais (). Cada pontuação exibindo um minigráfico na barra lateral direita quando selecionada. As grades podem ser classificadas por coluna em ordem ascendente ou descendente, bastando para isso clicar no cabeçalho da coluna.

É possível exibir diversos blocos de Seleção com pontuações para váriascombinações de Canal+Medida . Também é possível aplicar qualquer filtro disponível do Portal Analítico.. Cada bloco de Seleção pode ser exibido em uma grade ou em células. Quando o novo bloco tiver sido gerado, você poderá fazer as revisões necessárias. Além disso, você pode usar o botão Fixar () para construir um novo painel ou adicionar o bloco a um painel existente.

Smoothing

You can also apply Smoothing to a trend line to compress, or lessen the severity of changes. The Smoothing feature averages scores across the time span you select.

The Smoothing feature allows you to reduce the severity of the trend line between highs and lows. You can select the level of smoothing in the Settings sidebar (). In the following image, the darker trend line has One Month applied to a Satisfaction Quarterly report. The system has averaged each day's Satisfaction score with the scores of the preceding and succeeding 15 days. A Two weeks application would average each score with scores of the seven days before and seven days after, etc. The Best Fit smoothing is the average of a static period starting from the first day of the period, such as the first day of the week or (in this example) month.
