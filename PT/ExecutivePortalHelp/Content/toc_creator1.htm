O Criador de Painéis desenvolve e organiza blocos personalizados em painéis a fim de apresentar os dados ao Visualizador de Painéis. Esta seção da Ajuda é específica à função Criador de Painéis.

Navegação lateral

Use o conjunto de ícones no lado esquerdo do aplicativo para visualizar ou criar painéis e blocos.

detalhes...

A navegação lateral é um recurso baseado em permissão exibido ao longo do lado esquerdo da tela com ícones para passar de um ambiente para outro com um simples clique. O principal propósito desses ambientes é gerenciar os painéis para apresentar os dados. Cada ambiente é descrito sucintamente na tabela a seguir com um link para informações adicionais.

A trilha de navegação é exibida abaixo da barra de ferramentas como referência ao ambiente e item que está sendo visualizado. Por exemplo: Se você estiver visualizando o Painel ABC, a trilha de navegação exibirá: Painéis > ABC.

Cada ambiente oferece um menu de uma ou mais opções dentro do ambiente. O menu é exibido ao lado da navegação lateral. é preciso fazer uma seleção no menu a fim de mudar para o ambiente selecionado.

A tabela a seguir descreve cada elemento da navegação lateral:

Mostrar/Ocultar Cada ambiente contém um menu de opções para exibição. Clique no ícone Ocultar () para maximizar o modo de exibição.
Painéis Veja Como exibir painéis para obter informações adicionais.
Pontuações Veja Como trabalhar com pontuações para obter informações adicionais.
Perguntas abertas Veja Como trabalhar com perguntas abertas para obter informações adicionais.
Parâmetros de referência Veja Como trabalhar com parâmetros de referência para obter informações adicionais.
