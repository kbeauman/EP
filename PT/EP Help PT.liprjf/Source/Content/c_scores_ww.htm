Como trabalhar com pontuações

Detalhe os dados que compõem um bloco de Pontuações.

O Portal Executivo oferece dois tipos de pontuações:

  • Resumo de múltiplos canais - Uma pontuação combinada do Portal Analítico. Os dados podem ser exibidos em um gráfico de pizza ou grade tipo planilha e fixados em um painel.
  • Comparação de respostas - Os dados de uma única pergunta e medida específica. É possível exibir os dados em um gráfico de pizza, gráfico de barras verticais ou horizontais, ou grade tipo planilha. Essas opções de exibição podem ser fixadas a um painel.
Construir um novo bloco de pontuações...

Exibir uma pontuação combinada do Portal Analítico, ou as pontuações de perguntas específicas.

  1. Na navegação lateral, clique no ícone Pontuações () e selecione o ambiente desejado para a barra lateralPontuações.
    2. A página carrega um bloco padrão, que é o seguinte:
    • Intervalo de datas do dia anterior, ou seja, ontem.
    • Todos os menus configurados para o primeiro item.
  2. Selecione um intervalo de datas.
  3. Selecione os critérios (consulte Seleção e configuração dos dados para obter mais informações sobre os critérios dos blocos):
    • Resumo de múltiplos canais - Selecione um item no menu Pontuação combinada. Esses itens de menu são gerenciados em Portal Analítico > Multimedida > Pontuação combinada.
    • Comparação de respostas:
      1. Na barra lateral de Seleção de dados, selecione o Canal e a Medida desejados. Alguns itens do menu exibem menus de filtro adicionais, que são opcionais.
      2. (Opcional) Clique em Adicionar nova seleção para escolher outro Canal e Medida.
      3. Clique em Aplicar e, em seguida, clique no ícone Configurações (). As barra lateral Configurações será exibida.
      4. Selecione a Pergunta e a Pontuação desejadas para cada Medida.

Resumo de múltiplos canais

Por padrão, o painel de exibição mostra um bloco representando os dados do dia anterior para o primeiro item do menu Pontuação combinada na barra lateral Configurações. As pontuações são mostradas em um gráfico de pizza com as informações básicas no cabeçalho do bloco. Clique na divisa acima do bloco para mostrar ou ocultar o número de Entrevistados. Selecione um intervalo de datas e um item do menu Pontuação combinada na barra lateral Configurações para exibir outros resumos. Consulte a Ajuda do Portal Analítico para obter informações adicionais sobre pontuações combinadas. No Portal Analítico, clique no ícone de Ajuda e perguntas frequentes () e acesse Guias de Ajuda > Guia Multi-medida > Pontuação combinada.

Os seguintes modos de exibição estão disponíveis...

Exibição de gráfico de pizza

Passe o ponteiro do mouse sobre os elementos do gráfico para exibir pontuações menores. As etiquetas podem ficar ocultas quando várias pontuações têm valores mais baixos. Elas podem ser exibidas posicionando-se o ponteiro do mouse sobre cada elemento do gráfico.

Essa visualização interativa mostra a pontuação de cada canal com um minigráfico e a pontuação combinada no centro. Clique em cada um dos seguintes elementos para exibir os dados mais detalhados:

  • Etiqueta de pontuação do canal - Composto do canal, pontuação, e Contagem n, esse elemento exibe dados de Resumo, dos Elementos e dos Comportamentos futuros que compõem a medida.
  • Minigráfico - Exibe um gráfico de linhas com pontos de dados da pontuação total durante o intervalo de datas selecionado, juntamente com dados de Resumo que incluem a Pontuação, a Ponderação e a Contagem n.
    Minigráficos com duração inferior a 60 dias são mostrados em incrementos diários. Para períodos com mais de 60 dias, os minigráficos são mostrados em incrementos semanais.
  • Pontuação combinada - Composto do nome, pontuação, e Contagem n, esse elemento apresenta as mesmas informações que o minigráfico.

Exibição de grade

Esta tabela interativa oferece os seguintes recursos:

  • Pontuação combinada - Clique no cabeçalho da tabela para exibir dados de Resumo que incluem a Pontuação, a Ponderação e a Contagem n.
  • Classificar por coluna - Clique no cabeçalho de qualquer coluna para classificar os dados com base nos valores selecionados da coluna, com o menor valor primeiro. Clique novamente para inverter a ordem.
  • Pontuação do canal - Clique em cada item de linha para exibir os dados de Resumo, dos Elementos e dos Comportamentos futuros que compõem a medida. A divisa dupla do Resumo exibe a página do Portal Analítico correspondente.

Comparação de respostas

Estas pontuações são compiladas com base em diversos critérios selecionados nos menus das barras laterais Configurações e Dados. Por padrão, o painel de exibição mostra um bloco representando os dados do dia anterior para o primeiro item de cada menu. As pontuações são mostradas em um gráfico de pizza com as informações básicas no cabeçalho do bloco. Clique na divisa acima do bloco para mostrar ou ocultar dados adicionais. Selecione um intervalo de datas e os itens de menu desejados nas barras laterais Seleção de dados e Configurações.

Os seguintes modos de exibição estão disponíveis...
  • Gráfico de pizza - Cada seção (se houver espaço) é identificada com a resposta à pergunta, a pontuação e o percentual de respostas (Contagem n). Na imagem a seguir:
    • Resposta à pergunta = 5
    • Pontuação = 56
    • Percentual de respostas = 3%
    • Contagem n = 1

    No caso de haver várias respostas com Contagem n baixa, a etiqueta de cada uma delas só aparece quando o ponteiro do mouse é passado sobre o gráfico de pizza. Além disso, quando a resposta a uma pergunta estiver truncada, clique na etiqueta ou na seção de pizza para exibir toda a resposta na barra lateral.

  • Gráfico de barras verticais e gráfico de barras horizontais - Gráficos de barras são baseados na Contagem n, ou seja, a altura/comprimento de cada barra representa a porcentagem de respostas.
  • Grade -

Cada modo de exibição é interativo para exibir um minigráfico e dados de Resumo para cada item clicado.

Seleção de dados

Esta barra lateral apresenta menus de Canal e Medida, com a possibilidade de apresentar menus de filtro baseados nas suas seleções. Cada seleção exibe um bloco separado para fins de comparação. Quando uma Seleção é adicionada, um novo conjunto de Configurações é exibido.

Menus de filtro exibem um link de divisa dupla para Portal Analítico > Relatórios > Filtros de exibição.

Configurações

Cada Medida de seleção de dados tem um conjunto de menus correspondente para selecionar Pergunta e Pontuação.
