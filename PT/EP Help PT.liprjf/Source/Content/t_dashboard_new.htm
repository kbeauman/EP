Adicionar um painel

Cada bloco é exibido no painel na ordem em que foi adicionado. Por exemplo, o primeiro bloco adicionado está sempre no canto superior esquerdo do painel.

Existem dois métodos de construção de um novo painel:

  • Fixe um bloco a um novo painel - Clique no botão Fixar () e selecione Adicionar a novo painel na caixa de diálogo Fixar ao painel.
  • Usando o Gerenciador de Painéis - Este método permite configurar inicialmente a estrutura de um painel sem blocos. Use este método ao configurar um painel para outro proprietário, ao configurar as permissões de visualização e edição dos usuários ou como um espaço reservado para blocos ainda não selecionados.
