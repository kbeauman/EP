Compartilhar um painel

Clique no ícone Compartilhar () para enviar um e-mail com um link para visualização de um painel como uma página da Web ou para baixar uma imagem em PDF do painel. Use o Gerenciador de Painéis para visualizar um relatório em um painel compartilhado e para cancelar um compartilhamento.

Visualizar relatório compartilhado e cancelar um compartilhamento

Um relatório com os detalhes de um compartilhamento está disponível para Criadores de Painéis mediante a visualização das propriedades do painel a partir do Gerenciador de Painéis. Acesse Ferramentas > Gerenciador de Painéis, clique no nome do painel e abra o painel Compartilhar da caixa de diálogo para visualizar o relatório. Há também um botão Cancelar compartilhamento para desativar o link para a página da Web ou PDF.
