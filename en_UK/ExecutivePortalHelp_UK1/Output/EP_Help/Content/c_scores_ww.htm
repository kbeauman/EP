Working with Scores

Summary: Drill down into the data that comprises a Scores tile.

A Score is a numeric value, ranging from zero to one hundred, which represents a specific measure. Executive Portal offers two types of scores:

  • Multi Channel Summary - A Combined Score from Analytics Portal. You can view data in either a pie chart or spreadsheet type grid and pin it to a dashboard.
  • Response Comparison - Data from a single question and specific measure. You can view data in a pie chart, vertical or horizontal bar chart, or spreadsheet type grid. These viewing options may be pinned to a dashboard.
Build A New Score Tile...

Summary: Display a Combined Score from Analytics Portal, or the scores of specific Questions.

Role-based content: This section contains information and/or instructions requiring role-based features. If you do not see a feature described in this section, you may not have the necessary user profile to perform these tasks.

  1. On the side navigation, click the Scores icon () and select the desired environment from the Scores sidebar.
    2. The page loads a default tile, which is:
    • Date range of previous day, i.e., yesterday.
    • All menus set to first item.
  2. Select a date range.
  3. Select criteria (see Data Selection and Settings for information on tile criteria):
    • Multi Channel Summary - Select an item from the Combined Score menu. These menu items are managed in Analytics Portal > Multi Measure > Combined Score.
    • Response Comparison:
      1. On the Data Selections sidebar, select the desired Channel and Measure. Some menu items display additional filter menus, which are optional.
      2. (Optional) Click Add New Selection to choose another Channel and Measure.
      3. Click Apply and then click the Settings icon (). The Settings sidebar displays.
      4. Select the desired Question and Score for each Measure.
  4. View the tile in each display option to determine how the data is best communicated. Another option is to display a chart tile along with a grid tile of the same data.

When the new tile has been generated, you can make any revisions necessary. Also, you can use the Pin button () to build a new dashboard or add the tile to an existing dashboard.

Multi Channel Summary

By default, the display pane loads a tile representing the previous day's data for the first Combined score menu item in the Settings sidebar. Scores show in a pie chart with basic information in the tile header. Click the chevron above the tile to show or hide the number of Respondents. Select a date range and a Combined score menu item in the Settings sidebar to view other summaries. Refer to Analytics Portal Help for additional information on combined scores. In Analytics Portal, click the Help & FAQ icon () and go to Help Guides > Multi-Measure Tab > Combined Score.

The following views are available...

Pie Chart View

TIP: Mouse over chart elements to view lower number scores. When several scores have lower values the labels may be hidden. You can view these labels by positioning the cursor over each chart element.

This interactive view shows the score for each channel with a sparkline and Combined Score in the center. Click each of the following elements to view a drill-down of the data:

  • Channel Score Label - Consisting of the channel, score, and n Count, this element displays the Summary, Elements, and Future Behaviors data which make up the measure.
  • Sparkline - View a line chart with data points of the total score during the selected date range, along with Summary data consisting of Score, Weight, and n Count.
    NOTE: Sparklines of less than a 60 day duration are shown in daily increments. For periods exceeding 60 days, sparklines are shown in weekly increments.
  • Combined Score - Consisting of the name, score, and n Count, this element displays the same information as the sparkline.

Grid View

This interactive table offers the following features:

  • Combined Score - Click the table header to view Summary data consisting of Score, Weight, and n Count.
  • Sort-by-column - Click any column header to sort the data by the selected column values with the lowest value first. Click again to reverse the order.
  • Channel Score - Click each line item to view the Summary, Elements, and Future Behaviors data which make up the measure. The Summary double-chevron displays the corresponding Analytics Portal page.

Response Comparison

These scores are compiled by a number of selected criteria from menus of the Settings and Data sidebars. By default, the display pane loads a tile representing the previous day's data for the first item of each menu. Scores show in a pie chart with basic information in the tile header. Click the chevron above the tile to show or hide additional data. Select a date range and the desired menu items in the Data Selection and Settings sidebars.

The following views are available...
  • Pie Chart - Pie Charts are based on n Count, i.e., each slice is proportioned to reflect the distribution percentage of responses having a specific rating, with the lowest n Count at the top and increasing percentages progressing clockwise. Each section (space permitting) is labeled with the question response, score, and percent of responses (n Count). In the following image:
    • Question response = 5
    • Score = 56
    • Percent of responses = 3%
    • n Count = 1

    When several responses have a low n Count, the label for each only displays when you mouse-over the pie chart section. Also, when a question response is truncated, click the label or pie section to view the entire response in the sidebar.

  • Vertical Bar Chart and Horizontal Bar Chart - Bar Charts are based on n Count, i.e., each bar's height/length represents the percentage of responses. Ratings are sorted by the response options for the selected Question, e.g., a Question having response options of Yes, No, and Not Sure shows ratings in this same sequence.
  • Grid - Ratings are sorted by the response options for the selected Question, e.g., a Question having response options of Yes, No, and Not Sure shows ratings in this same sequence.

Each view is interactive to display a sparkline and Summary data for each item clicked.

Data Selection

This sidebar offers Channel and Measure menus, with the possibility of filter menus based on your selections. Each selection displays a separate tile for comparison. When you add a Selection, a new set of Settings display.

Filter menus display a double-chevron link to Analytics Portal > Reporting > View Filters.

Settings

Each Data Selection Measure has a corresponding set of menus to select Question and Score.
