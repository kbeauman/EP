Partager un tableau de bord

Sommaire: Cliquez sur l'icône Partager () pour envoyer un courriel ayant un lien pour consulter un tableau de bord comme une page Web ou pour télécharger une image PDF du tableau de bord. Utilisez le Gestionnaire de tableau de bord pour consulter un rapport sur le tableau de bord partagé et pour annuler un partage.

NOTE: Lorsque vous partagez un tableau de bord, le destinataire voit une copie virtuelle contenant des données qui sont gelées au moment où vous cliquez sur l'icône Partager. La version de la page Web d'un tableau partagé est encore interactive, mais les données ne changent pas.

Contenu basé sur les rôles: Cette section contient des renseignements et/ou des instructions nécessitant des fonctions basées sur les rôles. Si vous ne voyez pas une fonction décrite dans cette section, vous n'avez peut-être pas le profil d'utilisateur nécessaire pour effectuer ces tâches.

  1. L'icône Partager est située dans coin inférieur droit de la fenêtre de votre navigateur lorsque vous consultez un tableau de bord. Cliquez sur l'icône pour afficher la barre latérale Partage.
  2. Sélectionnez l'option souhaitée:
    • Lien pour le tableau de bord - Le destinataire peut consulter le tableau de bord comme une page Web. Cependant, les données ne changent pas à partir du moment où vous cliquez sur l'icône Partager.
    • Lien pour télécharger le PDF - Le destinataire peut sauvegarder le tableau de bord au format PDF. Chaque mosaïque de tableau de bord est placée sur une page séparée avec toutes les métadonnées affichées, c.-à-d. que l'en-tête est élargi. Cependant, les données d'extraction ne sont pas disponibles en PDF.
  3. Sélectionnez la période d'expiration du tableau de bord.
  4. (Optionnel) Saisissez un mot de passe dans le champ de texte pour empêcher l'accès à la page Web ou au PDF aux visiteurs non autorisés. Ce mot de passe n'est pas inclus dans le courriel et peut être transmis à un destinataire si vous le souhaitez.
  5. Cliquez sur Partager un tableau de bord. La barre latérale se ferme est une boîte de dialogue apparaît avec deux options:
    • Copier l'URL - Mettez en évidence la chaîne entière dans la zone texte et copiez (cliquez avec le bouton droit de la souris sur le texte mis en évidence et sélectionnez Copier dans le menu contextuel ou appuyez à la fois sur les touches Ctrl et C). Vous pouvez coller le lien souhaité.
    • Créer un courriel - Cliquez sur Créer un courriel pour que votre application de courriel par défaut affiche une note contenant l'URL. Saisissez l'adresse du destinataire et modifiez la note si nécessaire avant de l'envoyer.
  6. Pour fermer la boîte de dialogue, cliquez sur Annuler.

Consulter un rapport de partage et annuler un partage

Un rapport détaillé d'un partage est disponible pour les Créateurs de tableaux de bord en consultant les propriétés du tableau de bord à partir du Gestionnaire de tableau de bord. Allez à Outils > Gestionnaire de tableau de bord, cliquez sur le nom du tableau de bord et ouvrez le panneau Partager de la boîte de dialogue pour consulter ce rapport. Il y a également un bouton Annuler le partage pour désactiver le lien vers la page Web ou le PDF.
