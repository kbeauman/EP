Créer un nouveau tableau de bord avec le Gestionnaire de tableau de bord

Pour consulter le Gestionnaire de tableaux de bord, cliquez sur l'icône Outils () dans la navigation latérale et sélectionnez Gestionnaire de tableaux de bord dans la barre latérale Outils. La page Gestionnaire de tableaux de bord affiche une liste de tableaux de bord.

Cliquez sur Créer un nouveau tableau de bord. La boîte de dialogue Créer un nouveau tableau de bord apparaît avec plusieurs modèles de disposition et un champ de texte pour saisir le nom du nouveau tableau de bord.
