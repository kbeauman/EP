Utilisation de la page de connexion

Sommaire:  

  • Accéder au Portail exécutif ou au Portail analytique
  • Modifier le mot de passe
  • Modifier la langue de l'interface utilisateur
  • Soumettre des commentaires

Le Portail exécutif offre une fonction de connexion sécurisée qui est synchronisée avec le Portail analytique afin de faciliter une seule connexion pour les deux environnements. Votre session se ferme automatiquement après 30 minutes d'inactivité.

NOTE: Laisser votre session sans surveillance permet à n'importe qui d'accéder à votre appareil pour consulter vos données. N'oubliez pas de fermer votre session ou de verrouiller votre appareil lorsque vous vous absentez durant de longues périodes.

L'écran Connexion vous permet de sélectionner le portail que vous souhaitez consulter en premier. La navigation entre les portails est disponible sur une seule session de connexion.

Pour se connecter...
  1. Sélectionnez le portail souhaité dans le menu.
  2. Saisissez votre nom d'utilisateur et votre mot de passe.
  3. Si vous le souhaitez, sélectionnez la langue de votre choix dans le menu. Vous pouvez afficher le Portail exécutif dans les différentes langues indiquées. Cependant, le Portail analytique est seulement disponible en anglais.
  4. Cliquez sur Ouvrir une session. La page Tableaux de bord s'affiche.
Suggestion: Il est recommandé de modifier régulièrement votre mot de passe pour maintenir un haut niveau de sécurité.
Pour modifier ou réinitialiser votre mot de passe...
  1. Cliquez sur Mot de passe oublié. La boîte de dialogue Réinitialiser le mot de passe s'affiche.
  2. Saisissez votre adresse électronique et connectez-vous avec votre nom d'utilisateur puis cliquez sur Soumettre. La boîte de dialogue se ferme et un courriel est envoyé à l'adresse saisie avec de plus amples instructions.
Les autres fonctions disponibles dans la page Connexion comprennent...
Fonction Description
Menu Langues Réglez le Portail exécutif pour l'afficher dans un dialecte disponible.
Commentaires Affichez un courriel pré-adressé pour signaler un problème avec l'application ou fournir des suggestions pour des versions ultérieures.
