Au sujet du Gestionnaire de tableaux de bord

Allez à Outils > Gestionnaire de tableaux de bord pour accéder aux renseignements sur tous les tableaux de bord.

Le Gestionnaire de tableaux de bord est une grille ayant une fonction de triage par colonne qui est appelée en cliquant sur l'en-tête de colonne. Cette grille est triée par la colonne sélectionnée en ordre croissant. Cliquez à nouveau sur l'en-tête pour inverser l'ordre.

Consulter un tableau de bord

Cliquez sur l'icône Consulter pour afficher le tableau de bord. Le lien Revenir s'affiche dans la barre latérale vous permettant de fermer le tableau de bord et de revenir à la grille Gestionnaire de tableaux de bord.
Titre L'étiquette informative donnée au tableau de bord lorsqu'il a été créé.
État

Les tableaux de bord sont seulement visibles lorsqu'ils sont Activés. Désactivez un tableau de bord pour le mettre hors ligne sans le supprimer.

  2. Par défaut, le conteneur Général s'agrandit. Cochez ou effacez la case à cocher Activé.

Mosaïques Le nombre de mosaïques attribuées au tableau de bord.
Utilisateurs

Le nombre d'utilisateurs ayant l'autorisation de consulter ou de modifier le tableau de bord.

Pour gérer les affectations des utilisateurs...
  2. Cliquez sur la barre de titre Utilisateurs. Le conteneur s'agrandit.
  3. Faites défiler la grille Utilisateurs attribués et cochez ou effacez les cases Consulter et Modifier souhaitées. Sinon, utilisez le menu Copier les utilisateurs attribués à partir de pour appliquer les attributions à un autre tableau de bord.
  4. Cliquez sur Sauvegarder. La boîte de dialogue se ferme et la grille s'actualise pour refléter vos modifications dans la colonne Utilisateurs.
Propriétaire L'utilisateur qui a l'autorisation exclusive de supprimer le tableau de bord. Le propriétaire par défaut est l'utilisateur créant le tableau de bord.
Modifié Un timbre dateur généré par le système de la plus récente révision du tableau de bord.
Carrousel Utilisez la colonne Carrousel pour établir l'ordre dans lequel les tableaux de bord apparaîtront dans la barre latérale. Par défaut, le premier tableau de bord se charge lorsque vous vous connectez ou que vous naviguez dans la page Tableaux de bord. Cette séquence est pour votre profil d'utilisateur uniquement. Le meilleur moyen est de sélectionner un tableau de bord à chargement rapide comme no 1 afin d'éviter les délais de chargement lorsque vous consultez la page Tableaux de bord. Pour modifier la séquence du carrousel...Cliquez sur la zone de texte Carrousel souhaitée.

  1. Lorsque les nombres changent au milieu de la séquence, uniquement les tableaux de bord concernés par votre révision sont renumérotés. Ainsi, si vous souhaitez permuter les tableaux de bord n° 1 et n° 2, vous devez seulement remplacer le n° 1 avec le n° 2. Le n° 2 actuel est automatiquement remplacé par le n° 1 et les autres tableaux de bord restent inchangés. À l'inverse, remplacer le numéro le plus élevé par le n° 1 entraîne l'augmentation de tous les nombres de 1. Appuyez sur la touche Tabulation de votre clavier pour appliquer la modification.
  2. L'écran est actualisé pour afficher votre révision. Vous pouvez seulement modifier un numéro à la fois.

Lorsque vous cliquez sur une deuxième zone de texte, la première révision s'applique automatiquement.
Supprimer
Cette action entraîne l'élimination totale du tableau de bord sélectionné. Il est impossible de récupérer un tableau de bord supprimé.
Pour supprimer un tableau de bord...
  1. Sélectionnez la case à cocher dans la colonne Supprimer pour le tableau de bord. L'action Supprimer le(s) tableau(x) de bord apparaît au-dessus de la grille.
  2. Cliquez sur Supprimer le(s) tableau(x) de bord. Une boîte de dialogue apparaît vous demandant de confirmer l'élimination du tableau de bord.
  3. Cliquez sur Ok. La boîte de dialogue se ferme et l'écran s'actualise pour indiquer que la grille ne contient plus le tableau de bord supprimé.
À propos de la boîte de dialogue du Gestionnaire de tableaux de bord...

La grille contient tous les tableaux de bord auxquels vous avez accès ainsi que certains renseignements généraux.

Cliquez sur un élément pour afficher et réviser les détails de tableaux de bord suivants:

  • Général
    • Tableau de bord - Un champ de texte pour modifier le nom du tableau de bord.
    • État - Sélectionnez Activé (par défaut) pour que le tableau de bord soit actif.
    • Mosaïques - Une liste de mosaïques présentement attribuées au tableau de bord.
    • Disposition - Modifiez le modèle de disposition au besoin.
  • Utilisateurs
    • Propriétaire du tableau de bord - Un menu d'utilisateurs valides. Par défaut, le Propriétaire du tableau de bord a deux privilèges: Consulter et Modifier.
    • Copier les utilisateurs attribués à partir de - Appliquez les attributions d'utilisateurs à un autre tableau de bord.
    • Attributions d'utilisateurs - Sélectionnez les privilèges Consulter et Modifier pour chaque utilisateur.
  • Partager
    • Rapports partagés - Consultez les noms des utilisateurs qui ont partagé le tableau de bord ainsi que la date et la date d'expiration. Vous pouvez également annuler le partage.
Utiliser le Gestionnaire de tableaux de bord pour créer un nouveau tableau de bord...
  2. Sélectionnez le modèle de disposition souhaité, saisissez le nom dans Nom du tableau de bord et cliquez sur Sauvegarder. Le dialogue se ferme pour afficher le nom du nouveau tableau de bord dans le Gestionnaire de tableaux de bord.
  3. (Optionnel) Cliquez sur le nouveau nom pour consulter et modifier les propriétés du tableau de bord qui comprend les éléments suivants:
    • État - Activé par défaut.
    • Mosaïques - Une liste de mosaïques présentement attribuées au tableau de bord.
    • Disposition - Modifiez le modèle de disposition au besoin.
    • Propriétaire du tableau de bord - Un menu d'utilisateurs valides. Par défaut, le Propriétaire du tableau de bord a deux privilèges: Consulter et Modifier.
    • Attributions d'utilisateurs - Vous pouvez copier les attributions provenant d'un autre tableau de bord ou sélectionnez des privilèges dans la liste.
    • Rapports partagés - Consultez les noms des utilisateurs qui ont partagé le tableau de bord ainsi que la date et la date d'expiration. Vous pouvez également annuler le partage. Les tableaux de bord partagés demeurent dans ce rapport pendant 60 jours après l'expiration.
  4. Sélectionnez une icône du Groupes de mosaïques souhaité dans la navigation latérale et sélectionnez un élément de listes. Les groupes de mosaïques et les éléments de listes comprennent:
    • Résultats - Sommaire des méthodes multiples et comparaison de réponses
    • Questions ouvertes - Examen des commentaires
    • Repères - Repères standards
  5. Cliquez sur le bouton Épingler () de la mosaïque souhaitée. La boîte de dialogue Épingler au tableau de bord apparaît.
  6. Sélectionnez le tableau de bord souhaité et cliquez sur Appliquer. La boîte de dialogue se ferme et la mosaïque s'affiche à nouveau.
