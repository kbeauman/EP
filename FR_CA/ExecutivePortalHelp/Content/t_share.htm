Partager un tableau de bord

Cliquez sur l'icône Partager () pour envoyer un courriel ayant un lien pour consulter un tableau de bord comme une page Web ou pour télécharger une image PDF du tableau de bord. Utilisez le Gestionnaire de tableau de bord pour consulter un rapport sur le tableau de bord partagé et pour annuler un partage.

Consulter un rapport de partage et annuler un partage

Un rapport détaillé d'un partage est disponible pour les Créateurs de tableaux de bord en consultant les propriétés du tableau de bord à partir du Gestionnaire de tableau de bord. Allez à Outils > Gestionnaire de tableau de bord, cliquez sur le nom du tableau de bord et ouvrez le panneau Partager de la boîte de dialogue pour consulter ce rapport. Il y a également un bouton Annuler le partage pour désactiver le lien vers la page Web ou le PDF.
